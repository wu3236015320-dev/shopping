# 购物商城系统 - 小白从零开始部署指南

> 本文档面向完全不懂技术的小白，手把手教你把项目部署到新电脑上并运行起来。

---

## 📋 前言：你需要准备什么？

### 1. 硬件要求（最低）
- 电脑：Windows 10/11 系统
- 内存：4GB 以上
- 硬盘：至少 2GB 可用空间
- 网络：能上网（下载软件用）

### 2. 软件清单（需要安装）
| 软件 | 用途 | 下载地址 |
|------|------|----------|
| JDK 8 | Java运行环境 | https://www.oracle.com/java/technologies/downloads/ |
| MySQL 8.0 | 数据库 | https://dev.mysql.com/downloads/installer/ |

**注意**：如果你不确定电脑有没有这些软件，先往下看安装步骤。

## 第一步：安装 Java（JDK）

### 1.1 检查是否已安装
打开命令提示符（按 `Win + R`，输入 `cmd`，回车），输入：
```cmd
java -version
```
- 如果显示 `java version "1.8.xxx"` 或更高版本 → ✅ 已安装，跳过这一步
- 如果显示 `'java' 不是内部或外部命令` → ❌ 需要安装

### 1.2 下载 JDK
1. 访问：https://www.oracle.com/java/technologies/downloads/
2. 找到 **Java 8 (JDK 8)** 或 **Java 11**
3. 选择 **Windows x64** 版本下载（约 180MB）
4. 双击安装包，一路点击"下一步"安装

### 1.3 配置环境变量（关键步骤）
安装完成后，需要告诉系统 Java 在哪里：

1. 找到 JDK 安装路径（通常是 `C:\Program Files\Java\jdk-1.8.xxx`）
2. 右键"此电脑" → 属性 → 高级系统设置 → 环境变量
3. 在"系统变量"区域：
   - 新建变量 `JAVA_HOME`，值为你的 JDK 路径
   - 找到 `Path` 变量，编辑 → 新建 → 输入 `%JAVA_HOME%\bin`
4. 一路点确定保存

### 1.4 验证安装
重新打开命令提示符，输入：
```cmd
java -version
```
看到版本号说明成功。

## 第二步：安装 MySQL 数据库

### 2.1 检查是否已安装
打开命令提示符，输入：
```cmd
mysql --version
```
- 显示版本号 → ✅ 已安装，跳过安装步骤
- 报错 → ❌ 需要安装

### 2.2 下载 MySQL
1. 访问：https://dev.mysql.com/downloads/installer/
2. 下载 **MySQL Installer for Windows**（第二个大的安装包，约 300MB）
3. 双击运行安装程序

### 2.3 安装步骤
选择 **"Server only"**（仅安装服务器）即可：

1. 选择安装类型 → **Server only** → Next
2. 执行安装 → Execute → 等待安装完成
3. 配置服务器：
   - Config Type: **Development Computer**（开发电脑）
   - Port: **3306**（默认）
   - 记住这个端口号！
4. 设置 root 密码（超级重要！）：
   - 建议设置为：`root`
   - 这个密码后面会用到，务必记住！
   - 确认密码 → Next
5. Windows 服务配置：
   - 勾选 **Configure MySQL Server as a Windows Service**（开机自动启动）
   - 服务名：**MySQL80**（默认）
   - Next → Execute → 完成

### 2.4 验证 MySQL 安装
1. 按 `Win + R`，输入 `services.msc`，回车
2. 在服务列表中找到 **MySQL80**
3. 查看状态：
   - 如果是"正在运行" → ✅ 成功
   - 如果是空白/停止 → 右键点击 → 启动

## 第三步：准备项目文件

### 3.1 获取项目
从旧电脑复制以下文件夹到新电脑：
```
shopping-master/     ← 整个项目文件夹
```

**复制到哪里？**
建议放到新电脑的 `D:\` 盘根目录下，例如：
```
D:\shopping-master\
```

### 3.2 检查项目完整性
确保以下文件夹都存在：
- ✅ `bin/` - 编译后的代码
- ✅ `db/` - 数据库脚本
- ✅ `src/` - 源代码
- ✅ `WebRoot/` - 网页文件

---

## 第四步：导入数据库

### 4.1 找到数据库脚本
在项目文件夹中找到：
```
db/shopping.sql
```

### 4.2 执行导入命令
打开命令提示符，进入项目目录：
```cmd
cd /d D:\shopping-master
```

然后执行（假设 MySQL 密码是 `root`）：
```cmd
mysql -u root -p < db\shopping.sql
```

**详细步骤**：
1. 按回车后，会提示输入密码
2. 输入你安装 MySQL 时设置的密码（如 `root`）
3. 按回车，等待执行完成
4. 没有报错说明成功

### 4.3 验证数据库导入
执行以下命令检查：
```cmd
mysql -u root -p -e "SHOW DATABASES;"
```
应该能看到 `shopping` 数据库。

## 第五步：启动项目

### 5.1 方法一：一键启动（推荐）
双击运行：
```
一键启动.bat
```

这个脚本会自动：
1. 检查 MySQL 是否运行（没运行会自动尝试启动）
2. 编译项目（如果需要）
3. 启动网站服务器
4. 打开浏览器访问 http://127.0.0.1:8090

**等待时间**：
- 首次启动可能需要 10-30 秒
- 看到浏览器打开就说明成功了

### 5.2 方法二：手动启动
如果一键启动失败，可以手动操作：

1. **确保 MySQL 运行**：
   - `Win + R` → `services.msc`
   - 找到 MySQL80 → 右键启动

2. **编译项目**（首次需要）：
   - 双击 `编译.bat`
   - 等待显示"编译成功"

3. **启动网站**：
   - 双击 `启动项目.bat`
   - 看到"Starting Complete"说明成功

4. **打开浏览器**：
   - 访问 http://127.0.0.1:8090

## 常见问题排查

### ❌ 问题1：启动时显示 "503 Service Unavailable"

**原因**：MySQL 没启动或连不上

**解决**：
1. `Win + R` → `services.msc`
2. 找到 MySQL80 → 右键 → 启动
3. 重新运行 `一键启动.bat`

### ❌ 问题2："'java' 不是内部或外部命令"

**原因**：Java 环境变量没配置好

**解决**：
1. 检查 JDK 是否安装
2. 重新配置 JAVA_HOME 环境变量
3. 重启命令提示符再试

### ❌ 问题3：数据库导入失败 "Access denied"

**原因**：MySQL 密码错误

**解决**：
```cmd
mysql -u root -p
# 输入正确密码
# 进入后执行：
source db/shopping.sql;
```

### ❌ 问题4：商品图片不显示

**原因**：图片文件没复制过来

**解决**：
1. 检查 `WebRoot/img/product/` 文件夹
2. 应该有 1.jpg ~ 70.jpg 等图片
3. 如果没有，从旧电脑重新复制

### ❌ 问题5：网站打开很慢或卡顿

**原因**：可能是 Java 内存不足

**解决**：
1. 关闭其他占用内存的程序
2. 重启电脑后再试
3. 确保电脑有 4GB 以上内存
