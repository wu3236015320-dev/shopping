# 网上商城 — 部署说明（小白版）

按下面顺序做，就能在本机跑起来。每步都写了「可能遇到的情况」，照着做即可。

**项目根目录**（下面会反复用到）：即本仓库根目录，例如 `d:\003_fwork\cc\shopping-master`。  
若你的项目不在这个路径，把文档里所有路径改成你自己的。

---

## 第一步：准备好 MySQL

### 1.1 安装 MySQL（还没装的话）

- 打开浏览器访问：https://dev.mysql.com/downloads/mysql/
- 选 **Windows (x86, 64-bit), MSI Installer** 下载，安装时**把 root 密码设成 `root`**（和项目默认一致，省事）。
- 安装完成后，若勾选了“Start MySQL Server after installation”，服务会自动启动。

### 1.2 确认 MySQL 在运行

- 按 **Win + R**，输入 **services.msc** 回车。
- 在列表里找到 **MySQL96** 或 **MySQL80**（看你装的版本），看“状态”是否为 **正在运行**。
- 若是“已停止”，右键 → **启动**。

---

## 第二步：用 SQL 建好数据库和表

项目需要先有一个叫 **shopping** 的数据库，做法是执行一次项目里的 **db\shopping.sql**。

### 方式 A：用 CMD 一条命令（推荐，最快）

1. 按 **Win + R**，输入 **cmd**，回车，打开命令提示符。
2. 输入下面这一行（整行复制），回车，进入项目根目录：
   ```bat
   cd /d d:\003_fwork\cc\shopping-master
   ```
3. 再输入下面这一行（把 `C:\Program Files\MySQL\MySQL Server 9.6\bin` 换成你电脑上 MySQL 的安装路径，若装的是 8.0 就改成 `MySQL Server 8.0`；把 `root` 换成你的 MySQL 密码）：
   ```bat
   "C:\Program Files\MySQL\MySQL Server 9.6\bin\mysql.exe" -u root -proot < db\shopping.sql
   ```
   **注意**：`-proot` 中间没有空格，`root` 就是密码。若密码是 123456，就写 `-p123456`。

**可能看到：**  
- 出现 **“Using a password on the command line interface can be insecure.”** → 这是**提示**，不是报错，说明 SQL 已经执行成功，可以忽略。
- 若提示 **“找不到 mysql.exe”** → 说明 MySQL 的路径不对，用下面的「方式 B」用图形界面执行。

### 方式 B：用 MySQL Workbench（图形界面，不会命令也能做）

1. 打开 **MySQL Workbench**（若没装，到 https://dev.mysql.com/downloads/workbench/ 下载安装）。
2. 在首页**双击**本机的 MySQL 连接（或点 + 新建连接：主机 127.0.0.1，端口 3306，用户名 root，密码填你设的）。
3. 菜单 **File** → **Open SQL Script...**，选项目里的 **db\shopping.sql**（完整路径：项目根目录\db\shopping.sql）。
4. 点工具栏的 **闪电图标**（或按 Ctrl+Shift+Enter）执行整份脚本，没报错即成功。左侧刷新后能看到 **shopping** 数据库。

### 若你的 MySQL 密码不是 root

- 用记事本或 IDE 打开项目根目录下的 **src\config.properties**，修改：
  ```properties
  jdbc.username=root
  jdbc.password=root
  ```
  改成你的 MySQL 用户名和密码，保存即可。

---

## 第三步：安装并配置 JDK 8

### 3.1 安装 JDK 8

1. 按 **Win + R**，输入 **powershell**，回车。
2. 在窗口里**整行复制**下面命令，粘贴后回车：
   ```powershell
   winget install AdoptOpenJDK.OpenJDK.8 --accept-package-agreements --accept-source-agreements
   ```
3. 等待安装完成（会下载约 90MB）。

**可能看到：**  
- **“找到已安装的现有包... 找不到可用的升级”** → 表示 JDK 8 **已经装好了**，不用再装，直接做 3.2。
4. 若刚装完，**关掉这个 PowerShell 窗口**（后面用新窗口才能用上新环境变量）。

### 3.2 配置环境变量（让系统能找到 java）

1. **新开一个** PowerShell 窗口。
2. 依次输入下面两行，每行回车一次（路径改成你的项目根目录）：
   ```powershell
   cd d:\003_fwork\cc\shopping-master
   .\setup_java_env.ps1
   ```
3. 若出现 **“找到 JDK: …”** 并显示 Java 版本，说明配置成功。
4. **重要**：配置好后**关掉所有已打开的 CMD、PowerShell、Cursor 窗口**，再重新打开，环境变量才会生效。

---

## 第四步：启动项目

1. 打开 **CMD**（Win + R → 输入 cmd → 回车）。
2. 输入：
   ```bat
   cd /d d:\003_fwork\cc\shopping-master
   ```
   回车。
3. 再输入：
   ```bat
   启动项目.bat
   ```
   回车。
4. 等几秒，出现 **“Starting web server on port: 8090”** 和 **“Starting Complete. Welcome To The JFinal World :)”** 就表示启动成功。
5. **这个 CMD 窗口不要关**，关掉网站就停了。

**可能看到：**  
- **log4j:WARN No appenders...** → 可以忽略，不影响使用。  
- **port: 8090 already in use!** → 说明 8090 被占用了（多半是上次启动没关）。处理办法：  
  - 关掉之前运行过 `启动项目.bat` 的 CMD 窗口；或  
  - 在新 CMD 里先执行下面这一行，再重新运行 `启动项目.bat`：  
    ```bat
    for /f "tokens=5" %a in ('netstat -ano ^| findstr :8090') do taskkill /F /PID %a
    ```
- **HTTP 503 / Service Unavailable**（浏览器打开 127.0.0.1:8090 显示 503）→ 多半是数据库没连上。请确认：MySQL 服务在运行、已执行过 **db\shopping.sql**、MySQL 密码与 **src\config.properties** 里一致（或未配置时与 ShopConfig 默认值一致）。

---

## 第五步：在浏览器里打开网站

- **前台首页**：http://127.0.0.1:8090  
- **后台管理**：http://127.0.0.1:8090/admin  
  - 账号：**admin**  
  - 密码：**admin**  

### 若首页 / 商品列表里看不到测试商品（只有 1 个或没有）

商品是从 **MySQL 的 shopping 库**里读的。若你之前只执行过一次 **db\shopping.sql** 且是旧版本，库里可能只有 1 条商品。

**做法：在已有数据库上补一批测试数据**

1. 在 CMD 里进入项目根目录后执行（把 MySQL 路径和密码换成你自己的）：
   ```bat
   "C:\Program Files\MySQL\MySQL Server 9.6\bin\mysql.exe" -u root -proot shopping < db\seed_products.sql
   ```
2. 或用 **MySQL Workbench**：打开 **db\seed_products.sql**，选择数据库 **shopping**，执行整份脚本。
3. 执行成功后，**刷新浏览器**：
   - 打开 **http://127.0.0.1:8090/**（根路径）可看到「最新商品」每页 12 个，支持分页；
   - 点导航里的 **特价商品 / 推荐商品** 可看分页的完整商品列表；
   - 左侧「商品分类」和顶部「导航分类」里选分类也可按类看商品。

**注意**：必须用 **http://127.0.0.1:8090/** 或 **http://127.0.0.1:8090** 访问首页，不要直接打开磁盘上的 index.html，否则没有数据。

## 步骤一览（可打勾自检）

- [ ] MySQL 已安装且服务正在运行，root 密码已知（建议设为 root）
- [ ] 已执行 **db\shopping.sql**（CMD 或 MySQL Workbench 任选一种）
- [ ] 若 MySQL 密码不是 root，已修改 **src\config.properties** 中的 jdbc.username / jdbc.password
- [ ] 已用 winget 安装 JDK 8（若提示“已安装”则跳过）
- [ ] 已在项目目录运行 **setup_java_env.ps1**，并重新打开了终端
- [ ] 在项目目录运行 **启动项目.bat**，看到 “Starting Complete” 且无数据库报错
- [ ] 浏览器能打开 http://127.0.0.1:8090 和 http://127.0.0.1:8090/admin  

全部打勾后，部署就完成了。若某一步报错，把**完整报错内容**复制下来，方便排查。
